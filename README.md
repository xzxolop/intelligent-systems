# intelligent-systems

В этом репозитории хранятся лабораторные по курсу "Интеллектуальные системы". ОЧЕНЬ краткое описание контента:

* `intelligent-systems` - (лаба 1) Поиск решения в пространстве состояний 
* `lab7_NeuralNetwork1` - Распознавание образов - 1
* `lab8_RecognStudents` - Распознавание образов - 2
* `lab9_NeuralNetrworkTLGBot` - Финальный проект в виде чат-бота для платформы Telegram (AIML и распознавание образов)

По-хорошему проект нужно пересоздать т.к. проекты lab7 - lab9 лежат в директории проекта intelligent-systems, что не есть хорошо. 

При запуске `intelligent-systems.sln` запустится только проект с поиском в постранстве состояний. Остальные проектыц нужно запускать отдельно.
